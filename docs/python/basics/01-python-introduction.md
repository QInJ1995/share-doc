# Python 简介

## 什么是 Python？

Python 是一种高级、解释型、通用的编程语言。它由 Guido van Rossum 于 1991 年创建，以其简洁的语法和强大的功能而闻名。

## Python 的特点

- **简单易学**：语法清晰，代码可读性强
- **跨平台**：支持 Windows、Linux、macOS 等操作系统
- **丰富的标准库**：内置大量实用的模块和函数
- **强大的社区支持**：拥有庞大的开发者社区和丰富的第三方库
- **多范式支持**：支持面向对象、命令式、函数式编程

## Python 的应用领域

- Web 开发
- 数据分析与科学计算
- 人工智能与机器学习
- 自动化运维
- 游戏开发
- 网络爬虫
- 桌面应用开发

## Python 的版本

目前 Python 主要有两个版本：
- Python 2.x（已停止维护）
- Python 3.x（当前主流版本）

建议使用 Python 3.x 版本进行开发。

## 安装 Python

1. 访问 Python 官网：https://www.python.org/downloads/
2. 下载对应操作系统的安装包
3. 运行安装程序，确保勾选"Add Python to PATH"
4. 验证安装：打开命令行，输入 `python --version`

## 第一个 Python 程序

让我们编写一个简单的 Python 程序：

```python
print("Hello, World!")
```

将上述代码保存为 `hello.py`，然后在命令行中运行：

```bash
python hello.py
```
